# Agent Session Viewer

A web viewer for browsing and searching AI agent coding sessions (Claude Code, with more agents planned).

## Quick Start

```bash
uv tool install agent-session-viewer
agent-session-viewer
```

Opens a browser with your synced sessions. Use `--port 9000` for a custom port or `--no-browser` to skip auto-open.

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `j` / `k` | Next / previous message |
| `]` / `[` | Next / previous session |
| `o` | Toggle message order |
